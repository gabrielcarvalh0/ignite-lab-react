GRAPHQL = EXISTEM DOIS TIPOS DE CONSUMIR UMA API REST

1. QUERY = PARA BUSCAR DADOS 

2. MUTATION = CRIAR, ALTERAR E DELETAR DADOS DA API

O IMPORTANTE DE USAR GRAPHQL:
VOCÊ CONSEGUE TRAZER DADOS COM RELACIONAMENTOS ENTRE SI, RETORNANDO DADOS QUE VOCÊ PRECISE APENAS

overfitting = BACKEND RETORNA MUITO MAIS INFOR QUE VOCÊ PRECISE 

underfitting = QUANDO O BACKEND NÃO RETORNA TODAS AS INFOR QUE PRECISAMOS NO FRONTS

É O FRONT QUE DIZ PARA O BACK QUAIS INFO QUE ELE PRECISA E O BACK SE ADAPTA A ISSO USANDO O GRAPHQL
